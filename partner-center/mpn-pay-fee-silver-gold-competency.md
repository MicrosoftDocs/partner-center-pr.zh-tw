# <a name="how-to-pay-the-fee-associated-with-a-silver-or-gold-competency"></a>如何支付與銀級或金級專長認證相關聯的費用

**適當的角色**

- MPN 合作夥伴系統管理員
- MPN 帳戶管理員

本文說明如何及何時支付獲得銀級或金級 Microsoft 合作夥伴網路專長認證的相關費用。 本文也會說明如何變更您慣用的付款方法、如何取消訂單，以及如何在從銀級轉移至金級專長認證時獲得25% 的費用折扣。

> [!NOTE]
> 若要深入瞭解取得銀級或金級專長認證的優點、可用的特定專長，以及其獨特的需求，請參閱 [Microsoft 合作夥伴網路](https://partner.microsoft.com/membership/competencies)專長認證。 您也可以查看 Microsoft 合作夥伴網路 [授權資料表的](https://assetsprod.microsoft.com/mpn-maps-software-iur-competency-license-table.docx) 專長認證，以深入瞭解每個可用專長認證的權益。

## <a name="meet-a-competencys-requirements-and-pay-the-fee"></a>符合專長認證的需求並支付費用

如果您已經符合要求以取得特定的銀級或金級專長認證 (或者您已經有銀級或金級專長認證) ，您可以快速檢查專長認證的費用。 只需選取合作夥伴中心的 [ [比較-](https://partner.microsoft.com/membership/compare-offers) 供應專案] 索引標籤。

否則，請遵循下列步驟來支付銀級或金級專長認證的費用。 這些步驟假設您已符合下列必要條件：

- 您是已 [加入](https://partner.microsoft.com/membership) Microsoft 合作夥伴網路的合作夥伴。

- 您已符合 [要求](https://partner.microsoft.com/membership/competencies) 以取得特定的銀級或金級專長認證。

- 您已獲指派 [全域管理員或 MPN 合作夥伴系統管理員](./permissions-overview.md)的合作夥伴中心角色。如果您是沒有其中一個角色的來賓使用者，您將會被拒絕存取以下所述的一些合作夥伴中心頁面。 如果發生這種情況，請 [洽詢您的全域管理員](./find-your-role.md) ，並要求他們將適當的角色指派給您。 

1. 使用您組織的公司帳號憑證，登入合作夥伴中心 [儀表板](https://partner.microsoft.com/dashboard)。

2. 從導覽功能表中選取 [ [成員資格提供](https://partner.microsoft.com/dashboard/mpn/offers) 者]。 

3. 在 [ **供應專案可供購買**] 底下，尋找與您的特定專長認證相關聯的銀級 **成員資格** 磚或 **金級成員資格** 。 

4. 選取位於特定銀級或金級成員資格磚右下角的 **訂閱** 選項。

   :::image type="content" source="images/offers/1-mpn-competency-payment-offers-available-for-purchase.png" alt-text="[成員資格提供] 頁面顯示合作夥伴可以訂閱或購買的銀級或金級成員資格。" border="false":::

   > [!Note]
   > 如果您尚未滿足所需銀級或金級專長認證的需求，您將無法訂閱銀級/金級成員資格或支付相關聯的專長認證費用。 此外，如果您已經是金級專長認證合作夥伴，將會自動為您提供銀級專長認證的增量權益。 在此案例中，支付銀級專長認證費用的選項也將無法使用。

4. 在 [ **購買詳細資料** ] 頁面上，檢查並確認您的詳細資料。

    (深入瞭解如何將 [公司的 **稅務識別碼** 資訊](./organization-tax-info.md) 或公司的 [**帳單寄** 送到公司位址](./mpn-get-action-pack.md)。 ) 

   :::image type="content" source="images/offers/2-mpn-competency-payment-preference.png" alt-text="成員資格提供購買詳細資料頁面與付款喜好設定。" border="false":::

5. 在頁面的左下方，找出 [ **選取您的付款喜好** 設定] 區段。 選取 [ **信用卡** ] 選項或 [ **電子銀行轉移** ] 選項，然後選取 **[下一步]**。 請參閱下列各節，以取得與任一個選項相關聯的其他步驟。

### <a name="use-a-credit-card-as-your-payment-preference"></a>使用信用卡作為您的付款喜好設定

請遵循下列步驟，以針對相關聯的銀級或金級專長認證成員資格完成信用卡付款。

1. 如果您選取 **信用卡** 作為慣用的付款方法，則會出現 [信用卡詳細資料] 頁面。 

   :::image type="content" source="images/offers/3-mpn-competency-payment-credit-card-details.png" alt-text="成員資格提供付款信用卡詳細資料頁面。" border="false":::

2. 選取靠近此頁面頂端的 **合約** 連結，並代表您的公司審查 MPN 供應專案購買條款與條件。 本合約提供有關計費帳戶、續約、退款資格和稅金的重要資訊。

3. 在相關欄位中輸入您的信用卡詳細資料和帳單資訊，然後選取 [ **付款**]。

   您的付款將會透過協力廠商的安全付款閘道進行收集。

   > [!NOTE]
   > 請確定您在此頁面上輸入的位址對應至信用卡上的位址。 如果合法的實體位址與信用卡的位址不同，請勿在此信用卡詳細資料頁面上使用您的法律實體位址。 此外，請確定郵遞區號正確，並對應至您在此頁面上輸入的位址。

一旦收到信用卡資訊且交易完成後，您的銀級或金級專長認證訂用帳戶就會立即啟用。

### <a name="use-electronic-bank-transfer-as-your-payment-preference"></a>使用電子銀行轉帳作為您的付款喜好設定

請依照下列步驟完成相關聯的銀級或金級專長認證成員資格的電子銀行轉移。

1. 如果您選取 [ **電子銀行轉移** ] 作為慣用的付款方法，則會看到 [ **審核購買詳細資料** ] 頁面。

   :::image type="content" source="images/offers/4-mpn-competency-payment-review-purchase-details.png" alt-text="成員資格提供電子銀行轉帳購買詳細資料頁面。" border="false":::

1. 選取此頁面底部的 **合約** 連結，並代表您的公司審查 MPN 供應專案購買條款與條件。 本合約提供有關計費帳戶、續約、退款資格和稅金的重要資訊。

1. 請參閱您將需要的銀行轉帳詳細資料。 

1. 在 **您選取的購買** 區域中，記下 Microsoft 的金額和特定的貨幣費用。 您將需要從您的銀行帳戶以該特定貨幣傳輸該金額。

1. 當您滿意所需的所有資訊時，請選取 [ **提交**]。

   > [!NOTE]
   > 當您成功 **提交** 訂單後，您的 **訂單號碼** 和 **MPN 識別碼** 就會出現。 請記下此資訊。 您稍後將需要此檔案才能完成銀行轉移。

1. 您將會看到確認頁面，指出您已成功提交。 當您完成與銀行的網路傳輸之後，您必須返回此頁面以上傳您的付款證明。

   :::image type="content" source="images/offers/5-mpn-competency-payment-thank-you-confirmation.png" alt-text="提交確認頁面。" border="false":::

1. 從您的銀行帳戶完成網路傳輸。 請務必執行下列動作：

   - 在電匯參考詳細資料中包含 **訂單號碼** 和 **MPN 識別碼** 。

   - 將先前所記下的金額 (以) 的貨幣傳輸至先前頁面的 [ **電匯指示** ] 區域中顯示的 Microsoft 銀行帳戶。

1. 當您的銀行確認交易之後，請將銀行確認相片視為證明。 請確定銀行確認證明檔會顯示下列詳細資料： 
   - 電匯付款的發生日期
   - 用來接收您電匯的 Microsoft 銀行帳戶
   - 以正確的貨幣顯示的傳輸數量
   - 您的訂單號碼和 MPN ID 號碼

1. 請參閱下一節，以瞭解上傳電匯付款檔的步驟。

### <a name="upload-proof-of-wire-transfer-payment"></a>上傳電匯證明付款

如果您透過電子電匯支付銀級或金級的成員資格費用，則必須將付款證明上傳至合作夥伴中心。 請遵循下列步驟來執行此動作：

1. 返回 [合作夥伴中心] [儀表板](https://partner.microsoft.com/dashboard)，然後從左側導覽功能表中選取 [ **成員資格** 供應專案] 選項。

1. 在 [購買的供應專案] 清單中，找出您的特定供應專案。 然後，在該供應專案的 [動作] 資料行下，選取 **[上傳付款證明**]。

   :::image type="content" source="images/offers/6-mpn-competency-payment-show-purchased-offers.png" alt-text="針對每個已購買的供應專案，顯示可選取選項、上傳付款證明的已購買成員資格供應專案清單。" border="false":::

1. 在 [詳細資料] 頁面上，尋找 [ **付款證明** ] 欄位。 選取 **[流覽]** 以找出您的證明檔，並將其上傳到合作夥伴中心。

   :::image type="content" source="images/offers/7-mpn-competency-payment-upload-proof.png" alt-text="詳細資料頁面，其中包含用來上傳顯示可選取選項的欄位、上傳付款證明，適用于每個購買的供應專案。" border="false":::

1. 針對與最近的電子資金轉帳交易相關的其他欄位，完成相關資訊，例如 **付款日期、付款****給帳戶號碼**，以及 **交易/Ref 否**。

  可能會出現綠色訊息，告訴您您的要求已成功完成。

在成功完成付款之後，您的購買通常會在48到72小時，在抵達 Microsoft 的銀行帳戶後顯示為使用中。 不過，完整程式最多可能需要 **一周** 的時間才能完成。 

如果您在電匯參考詳細資料中提供必要資訊 (例如，訂單號碼和 MPN 識別碼) ，並將付款證明上傳至合作夥伴中心，則可以更快地啟用付款。

如果付款程式未在訂單建立日期的30天內完全完成，訂單的狀態將會變更為「已 **過期**」。 如果您的訂單已 **過期**，您就必須直接從合作夥伴中心建立新訂單，而不需要再傳送一次付款。 

## <a name="switch-from-electronic-bank-transfer-payment-to-credit-card-payment-method"></a>從電子銀行轉帳付款條件切換至信用卡付款方法

您稍後可以決定變更與銀級或金級專長認證成員資格相關聯的付款方法。 例如，您可能會決定使用信用卡付款，而不是電子銀行轉帳。 或者，您可能已經在系統中建立 unactioned 的電子銀行傳輸訂單。

若要變更現有訂單的付款喜好設定，您必須先取消訂單，然後使用您慣用的付款方法來重新購買。

請執行下列動作來取消現有的 unactioned 電子銀行傳輸，並變更您的付款喜好設定：

1. 登入合作夥伴中心[儀表板](https://partner.microsoft.com/dashboard)。

2. 從左側導覽功能表中，選取 [ **成員資格提供** 者]。

3. 在 [購買的供應專案] 清單中，找出您想要取消的特定供應專案。 然後，在該供應專案的 [動作] 資料行底下，選取 [ **取消訂單**]。

   :::image type="content" source="images/offers/8-mpn-competency-payment-cancel-order.png" alt-text="購買的成員資格供應專案清單會顯示取消順序選項。" border="false":::

4. 遵循其餘的步驟來取消訂單。 

5. 請參閱先前的章節，以使用新的付款喜好設定來建立新的訂單，並根據您仍可使用的其中一個銀級或金級會員供應專案。

## <a name="apply-upgrade-discount-when-moving-from-silver-to-gold"></a>將升級折扣從銀級移至金級

如果您的組織有使用中的銀級專長認證，而且符合相關金級專長認證的資格需求，則您有資格從銀級升級為金級以獲得折扣。 金級專長認證的價格將獲得25% 的折扣。

新的金週年日將會排程為一年的購買日期。 您將獲得完整的黃金優惠。

> [!NOTE]
> 此升級折扣有一些例外狀況。 如果您有銀級的成員資格供應專案，則使用25% 金級會員折扣來升級為金級的成員資格供應專案，您將不會有銀級退款的資格。 即使您在30天內支付了銀級的成員資格，而且尚未使用相關權益，也是如此。

若要查看25% 折扣的套用位置，請執行下列動作：

1. 登入合作夥伴中心[儀表板](https://partner.microsoft.com/dashboard)。

2. 從左側導覽功能表中，選取 [ **成員資格提供** 者]。

3. 在 [供應專案可供購買] 底下，尋找 [金級成員] 磚。 如果您先前已獲得並支付銀級成員資格，則金級成員資格金額將會調整，以反映25% 的折扣。

   :::image type="content" source="images/offers/9-mpn-competency-silver-to-gold-discount.png" alt-text="購買的成員資格供應專案會顯示具有銀級的金級磚供應專案，價格為25%。" border="false":::

4. 選取反映折扣的金級成員資格磚內的 [ **訂閱** ]。 您的購買詳細資料頁面也會顯示所選購買的折扣。

   :::image type="content" source="images/offers/10-mpn-competency-silver-to-gold-discount-2.png" alt-text="詳細資料頁面會顯示有25% 折扣的供應專案。" border="false":::

5. 選取 **[下一步]** 以完成購買此供應專案並享有折扣。
