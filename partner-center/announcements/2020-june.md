## <a name="now-available-windows-7-year-2-esu-for-csp-partners"></a><a name="6"></a>現已推出適用於 CSP 合作夥伴的 Windows 7 Year 2 ESU

### <a name="categories"></a>類別

- 日期：2020-06-29
- 輸入：供應項目/市場
- 狀態：現已推出

### <a name="summary"></a>摘要

Windows 7，Year 2 延伸安全性更新 (ESU) 定價現已在[資源庫](https://partner.microsoft.com/resources/collection/windows-esu-csp#/)中。

### <a name="impacted-audience"></a>影響對象

參與 CSP 計畫的所有合作夥伴

### <a name="details"></a>詳細資料

支援在 2020 年 1 月 14 日結束的 Windows 7。 雖然許多客戶都有辦法部署 Windows 10，但 Microsoft 了解每個人都在其升級過程中的不同點。 Windows 7 Year 1 ESU 的有效期間將於 2021 年 1 月 12 日結束。 我們知道客戶正開始規劃下一年。  

對於預計在 2021 年繼續使用 Windows 7 裝置的客戶，我們現在將 Windows 7 Year 2 ESU 的定價提供給 CSP 合作夥伴，以便協助客戶進行 Windows 10 決策。 如需詳細資訊，請造訪[資源庫](https://partner.microsoft.com/resources/collection/windows-esu-csp#/)。

請與貴組織內適當的連絡人分享這項資訊。

## <a name="partner-center-relationship-removal-feature-update"></a><a name="5"></a>合作夥伴中心關聯性移除功能更新

### <a name="categories"></a>類別

- 日期：2020-06-29
- 輸入：功能
- 狀態：即將推出

### <a name="summary"></a>摘要

自 2020 年 6 月 29 日起，CSP 間接提供者和間接轉銷商可使用合作夥伴中心移除下游關係，而不再需要提交合作夥伴中心支援票證才能這麼做。 在執行這些活動時，這會為合作夥伴節省大量時間。

### <a name="impacted-audience"></a>影響對象

雲端解決方案提供者 (CSP) 間接提供者和間接轉銷商

### <a name="scenarios-supported"></a>支援的案例：

- 與間接轉銷商建立關聯性的間接提供者可以使用合作夥伴中心來移除該關聯性。
- 與終端客戶建立關聯性的間接轉銷商可以使用合作夥伴中心來移除該關聯性。

上述案例中的動作必須由合作夥伴中心的系統管理員代理程式執行。

### <a name="next-steps"></a>後續步驟：

- 檢閱作業整備資源庫集合資料：合作夥伴中心關聯性移除功能更新。
- 在貴組織內分享此資訊，以便適當的小組檢閱和更新流程。

## <a name="billing-frequency-default-changing-from-annual-to-monthly"></a><a name="4"></a>計費頻率預設從年度定期變更為每月

### <a name="categories"></a>類別

- 日期：2020-06-17
- 輸入：功能
- 狀態：即將推出

### <a name="summary"></a>摘要

在合作夥伴中心購買體驗中，計費頻率預設值從年度變更為每月。

### <a name="impacted-audience"></a>影響對象

雲端解決方案提供者 (CSP) 直接提供者和間接提供者

### <a name="details"></a>詳細資料

計費頻率設定會決定針對合作夥伴已購買的服務向其計費的頻率。 合作夥伴在合作夥伴中心目錄中購買時，會在**檢閱**步驟中看到這項設定。 此設定預設為**年度**，因為已針對授權型服務引進年度計費。 根據合作夥伴的意見反應，合作夥伴中心小組從 6 月 17 日起，將此預設值從**年度**變更為**每月**，以便更符合合作夥伴預期的情況。 合作夥伴可以在檢閱頁面上提交交易之前，或是[在從管理訂用帳戶頁面中提交交易之後]( https://docs.microsoft.com/partner-center/billing-annual-monthly#change-billing-frequency)，變更這項設定。 請注意，僅支援年度計費的供應項目，其預設值將持續為年度。 此預設變更僅適用於合作夥伴中心入口網站使用者體驗，合作夥伴中心 API 的運作方式不會變更。

我們希望這個變更能降低合作夥伴意外在無意中購買每年計費訂閱的可能性。

## <a name="changes-to-microsoft-365-f1"></a><a name="3"></a>Microsoft 365 F1 的變更

### <a name="categories"></a>類別

- 日期：2020-06-15
- 輸入：供應項目/市場
- 狀態：現已推出

### <a name="summary"></a>摘要

Microsoft 365 F1 將會以新的服務更新。 新的供應項目將於 6月 15 日發行以取代前一個 F1。

### <a name="impacted-audience"></a>影響對象

雲端解決方案提供者 (CSP) 直接提供者和間接提供者

### <a name="details"></a>詳細資料

自 2020 年 4 月起，合作夥伴中心已提供 Microsoft 365 F1。 藉由新增新服務的支援來增強功能：Exchange Kiosk 和 MS 搜尋功能。 新服務將會根據 6 月底的原始 F1 供應項目，在任何現有的訂用帳戶上自動佈建。 原始 F1 供應項目將於 6 月 15 日取代為新訂閱購買的新 F1 供應項目 (下方)。

原始 F1 供應項目：

   |**供應項目名稱**|**優惠識別碼**|**材質識別碼**|**佈建識別碼**|
   |-------------------|:------|:------|:------|
   |Microsoft 365 F1|1ce4a4ba-65f7-49e7-bb28-d92d07d58b86|1PI-00005|M365_F1|
   |Microsoft 365 F1 試用版|dd97e36a-f0df-4a9a-b461-625420656511|1PI-00006|M365_F1|

想要購買新 F1 訂閱的合作夥伴，應該在 6 月 15 日之前使用下列供應項目識別碼。 合作夥伴可以在合作夥伴中心目錄中看到這些識別碼。 運用合作夥伴中心 API 的合作夥伴也可以使用下列資訊。

   |**供應項目名稱**|**優惠識別碼**|**材質識別碼**|**佈建識別碼**|
   |-------------------|:------|:------|:------|
   |Microsoft 365 F1|17b4a1d6-2c65-478e-a046-51cf40d25d76|1PI-00012|M365_F1_COM|
   |Microsoft 365 F1 試用版|2e4e9424-06db-4d35-b2d8-3a1b0ef1dc63|1PI-00013|M365_F1_COMM|

### <a name="call-to-action"></a>喚起行動
具有目前 Microsoft 365 F1 產品的合作夥伴不用採取任何動作。 Exchange Kiosk 和 MS 搜尋的服務方案將會在 6 月底之前於現有服務方案中回填。

想要在 6 月 15 日之後購買 Microsoft 365 F1 的合作夥伴，應該使用新的 F1 供應項目資訊，以確保他們購買的是正確的供應項目。 新的 F1 供應項目識別碼和其他資訊將會在 7 月 1 日於七月份雲端解決方案提供者價目表更新中更新。

## <a name="new-and-end-of-life-microsoft-dynamics-365-and-power-platform-offers-and-products"></a><a name="1"></a>全新和生命週期結束 Microsoft Dynamics 365 和 Power 平台供應項目與產品

### <a name="categories"></a>類別

- 日期：2020-06-11
- 輸入：供應項目/市場
- 狀態：即將推出

### <a name="summary"></a>摘要

Microsoft 將在 2020 年 7 月 1 日發行新的 Dynamics 365 和 Power Platform 供應項目與產品，並淘汰適用的 Fraud Protection產品。

### <a name="impacted-audience"></a>影響對象

### <a name="details"></a>詳細資料

新的 Dynamics 365 Fraud Protection 產品：

- Microsoft Dynamics 365 Fraud Protection - 帳戶防護：本產品藉由偵測並防止詐騙活動，協助商家保護其線上客戶帳戶。
- Microsoft Dynamics 365 Fraud Protection - 外洩防護：使用此產品，零售商可以減少其實體商店的縮減。
- Microsoft Dynamics 365 Fraud Protection - 購買防護：本產品藉由偵測並防止詐騙活動，協助商家保護其線上交易。
- Microsoft Dynamics 365 Fraud Protection - 帳戶防護：本產品藉由偵測並防止詐騙活動，協助商家保護其線上客戶帳戶。

停止提供服務的 Microsoft Dynamics 365 Fraud Protection 產品：

下列 Microsoft Dynamics 365 Fraud Protection 供應項目將會從 2020 年 7 月 1 日價目表中移除：

- Microsoft Dynamics 365 Fraud Protection
- Microsoft Dynamics 365 Fraud Protection Addl Assessments
