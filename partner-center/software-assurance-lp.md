# <a name="participate-in-software-assurance-programs-including-training-vouchers-and-planning-services"></a>參與軟體保證計畫，包括訓練券和規劃服務

參與軟體保證計畫（包括軟體保證訓練券 (SATV) 和規劃服務），讓您有機會與 Microsoft 企業客戶拓展您的業務。 

- 作為軟體保證訓練券合作夥伴，Microsoft 將會補償您，以提供由 Microsoft 專家開發的講師指導、技術課堂訓練。 

- 作為規劃服務合作夥伴，Microsoft 將會補償您領導預先定義的實施計畫。 

您可以透過這兩種程式，讓您的業務與競爭對手分開設定、成為受信任的顧問，並提供客戶的後續服務。 程式中的註冊和相關軟體保證券的管理是透過合作夥伴中心完成。

> [!NOTE]
> 若要利用規劃服務供應專案及其特定參與專案，您需要符合各層級的專長認證需求。 這些專長認證需求包括在規劃服務供應專案層級完成訓練和其他需求，以及完成特定 engagement 本身的專長認證需求。  
>
