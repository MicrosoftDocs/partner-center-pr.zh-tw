# <a name="incentives-co-op-claims-core-requirements"></a>獎勵合作宣告需求

**適用於**

- 合作夥伴中心

**適當的角色**

- 獎勵管理員
- 獎勵使用者

您向其提交獎勵宣告的每一種活動類型都有核心需求。 您必須在執行證明文件中證明您符合這些需求。 有些活動還會需要特定的通訊類型和計量。

下表概述列出每個已接受活動的核心要求。 

## <a name="demand-generation"></a>產生需求

### <a name="print-advertising"></a>印刷廣告

| 類別 | 執行證明    |
| ------ | ----------- |
| 核心要求    | 公司品牌與連絡資訊<br><br>Microsoft 內容 |
| 通訊要求 | 最終通訊的複本 (根據廣告)，顯示計畫專屬核心要求 |
| 計量     | 預估客戶 (或合作夥伴) 觸及 |
|合格費用    | 第三方媒體置入<br><br>印刷材料<br><br>費用 - 代理商行銷和創意服務，包括代理商費用  |

### <a name="digital-advertising"></a>數位廣告

| 類別 | 執行證明    |
| ------ | ----------- |
| 核心要求 | NA |
| 通訊要求 | 數位廣告的螢幕擷取畫面<br><br>AND<br><br>登陸頁面的 URL，顯示計畫專屬的核心需求 |
| 計量     | 結果摘要 (例如，點選連結報告) |
|合格費用    | 第三方媒體置入<br><br>費用 - 代理商：行銷和創意服務，包括代理商費用 |

### <a name="direct-mail-email-and-short-message-service-sms"></a>直接寄件、電子郵件和短訊服務 (SMS)

| 類別 | 執行證明    |
| ------ | ----------- |
| 核心要求    | NA |
| 通訊要求 | 1) 顯示計畫專屬核心需求的最終通訊複本<br><br>2) 發佈證明可以包含蓋了郵戳的信封或其他郵寄證明 (直接寄件)、標頭顯示日期的最終電子郵件或其他寄送電子郵件證明 (電子郵件)，或手機訊息的相片 (簡訊) |
| 計量     | 預估客戶 (或合作夥伴) 觸及 |
|合格費用    | 資料庫收購<br><br>印刷材料<br><br>郵資<br><br>費用 - 代理商：行銷和創意服務，包括代理商費用 |

### <a name="partner-website-and-search-engine"></a>合作夥伴網站和搜尋引擎

| 類別 | 執行證明    |
| ------ | ----------- |
| 核心要求    | NA |
| 通訊要求 | 顯示 URL 的網站頁面的 URL 和/或螢幕擷取畫面<br><br>SEO 專案的工作或服務合約聲明 (僅限 SEO 活動) |
| 計量     | 結果摘要 (例如，點選連結報告) |
|合格費用    | 費用 - 代理商：行銷服務，包括代理商費用<br><br>程式設計/開發：程式設計和開發費用<br><br>費用 - SEO：最佳化和維護費用  |

### <a name="microsoft-syndicated-content"></a>Microsoft 同步發行內容

| 類別 | 執行證明    |
| ------ | ----------- |
| 核心要求    | NA |
| 通訊要求 | 顯示 URL 網站頁面的 URL 和/或螢幕擷取畫面 |
| 計量     | 結果摘要 (例如，點選連結報告) |
|合格費用    | 每個使用期間的實際成本高達 18,750 美元：費用 - 代理程式行銷服務，包括代理商費用<br><br>程式設計和開發費用<br><br>媒體置入  |

### <a name="multi-touch-digital-campaign"></a>多點觸控數位行銷活動

| 類別 | 執行證明    |
| ------ | ----------- |
| 核心要求    |NA |
| 通訊要求 | 需要在適用的活動頁面中詳細說明所有執行證明<br><br>數位廣告<br><br>直接傳送電子郵件<br><br>合作夥伴網站與搜尋引擎最佳化<br><br>電話行銷  |
