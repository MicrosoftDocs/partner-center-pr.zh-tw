# <a name="sell-software-subscriptions-through-the-cloud-solution-provider-csp-program"></a>透過雲端解決方案提供者 (CSP) 方案銷售軟體訂閱

**適用於**

- 雲端解決方案提供者

**適當的角色**

- 系統管理代理人
- 全域系統管理員

透過 Azure 保留和伺服器訂閱 (Windows Server 和 SQL Server 訂閱)，雲端解決方案提供者計畫中的合作夥伴將更能因應客戶快速增長的需求，以更具成本效益的解決方案支援具有高度可預測性與持續性的雲端工作負載。 

您現在可以運用 Azure Hybrid Benefit，透過合作夥伴中心和 Azure 入口網站，代表商業客戶取得、佈建和管理 Azure 保留和伺服器訂閱。

Azure Hybrid Benefit 有助於讓您的 Windows Server 授權發揮更大的效益，並可節省高達 40% 的虛擬機器。 您可以將權益搭配用於軟體保證所涵蓋的 Windows Server Datacenter 和 Standard Edition 授權。 根據版本，您可以轉換或重複使用在 Azure 中執行 Windows Server 虛擬機器的授權，並支付較低的基本計算費率 (例如 Linux 虛擬機器費率)。

## <a name="buy-software-subscriptions-on-behalf-of-customers"></a>代表客戶購買軟體訂閱

若要代表客戶購買軟體訂閱：

1. 登入合作夥伴中心[儀表板](https://partner.microsoft.com/dashboard)。

2. 從 [合作夥伴中心] 功能表上，選取 [客戶]，然後從清單中選擇客戶。

3. 在客戶的詳細資料頁面中選取 [新增產品]，然後依照畫面上的指示建立訂單並支付其費用。 除了澳洲和巴西以外，所有商業客戶定價都不含稅金。 澳洲和巴西的價格則包含稅金。

## <a name="activate-and-manage-software-subscriptions"></a>啟用及管理軟體訂閱

購買軟體訂閱後，請依照下列步驟加以下載。

>[!NOTE]
>您必須是系統管理代理人，才能下載軟體並取得啟用金鑰。

1. 登入合作夥伴中心[儀表板](https://partner.microsoft.com/dashboard)。

2. 從 [合作夥伴中心] 功能表上，選取 [客戶]，然後從清單中選擇客戶。

3. 移至客戶的詳細資料頁面，然後選取 [軟體]。 您會看到您已代表客戶購買的所有軟體清單。 

4. 展開您要下載的產品。 在 [選取產品] 欄位中，選取您想要的 [版本]、[語言] 和 [檔案類型/OS]。 

5. 選取 [提交] 以顯示特定產品。 

6. 選取 [取得金鑰和下載項目]。 

7. 選取 [下載] 以開始下載，或選取 [複製連結] 以複製連結並將其傳送給客戶。 

>[!NOTE]
>此連結會在兩個星期後或下載 50 次後到期 (視何者先發生)。 連結過期之後，請回到此頁面並再次選取 [取得金鑰和下載項目]，以再啟用另外兩週或 50 次下載。 您可以依需求執行此動作，次數不限。 

## <a name="server-subscription-download-and-license-keys-available-through-microsoft-365-admin-center-for-customers"></a>可以透過 Microsoft 365 系統管理中心為客戶提供伺服器訂閱下載和授權金鑰 

您的客戶能夠從 Microsoft 365 系統管理中心取得雲端解決方案提供者伺服器訂閱授權金鑰和下載。 若要查看其雲端解決方案提供者伺服器訂閱授權金鑰和下載，客戶必須前往 [Microsoft 365 系統管理中心] > [計費] > 您的產品 > [軟體] 索引標籤。如需詳細資訊，請參閱 [[計費] 底下的 [軟體] 索引標籤](https://docs.microsoft.com/microsoft-365/admin/whats-new-in-preview?view=o365-worldwide#billing--subscriptions)。  

## <a name="view-activity-for-software-key-access-and-software-downloads"></a>檢視軟體金鑰存取和軟體下載的活動

基於稽核或合規性目的，您可能需要檢查曾存取伺服器訂閱軟體金鑰或下載伺服器訂閱軟體的使用者清單。 請使用下列程序來存取這項資訊。 

>[!NOTE]
>您必須是全域管理員、帳戶管理員、推薦管理員或行銷內容管理員，才能看到這些活動記錄。 

1. 登入合作夥伴中心[儀表板](https://partner.microsoft.com/dashboard)。

2. 選取位於右上角的齒輪圖示。

3. 在功能表中，選取 [活動記錄]。

4. 輸入您要查看的活動所屬的日期範圍。 活動記錄會顯示在您指定的時間內曾存取軟體金鑰或下載軟體的使用者清單。 

## <a name="cancel-a-purchase"></a>取消購買

您可以在購買日期起算的 60 天內取消軟體購買。 如果您在這前 60 天的期間內取消購買，則無須支付提前終止費用。 在 60 天後，您就無法再取消購買。 (請參閱注意事項，以了解此取消規則的主要限制。 若要了解取消軟體購買的後續事項，請參閱下列步驟之後的重要注意事項。) 

>[!NOTE]
>下列關於取消購買的步驟僅適用於能夠在特定時間範圍內取消購買的軟體，例如在購買後的 60 天內。 這些步驟也不適用於 Azure 中的 SUSE Linux 或 RedHat 軟體方案。 目前，您無法取消或交換 SUSE 或 RedHat 軟體方案。 請[深入了解](https://docs.microsoft.com/azure/virtual-machines/linux/prepay-suse-software-charges) SUSE Linux 或 RedHat 方案的使用方式。

請依照下列步驟取消購買：

>[!NOTE]
>您必須是系統管理代理人，才能取消購買。 下列步驟說明如何在合作夥伴中心儀表板中取消購買。 您也可以使用[合作夥伴中心 API](https://docs.microsoft.com/partner-center/develop/cancel-software-purchases) 來執行此動作。

1. 開始進行取消程序之前，請確定您具有下列各項：

    - 客戶的名稱、租用戶 GUID 或網域名稱

    - 您要取消的產品名稱
    
    - 訂單識別碼

2. 登入合作夥伴中心[儀表板](https://partner.microsoft.com/dashboard)。

3. 從 [合作夥伴中心] 功能表上，選取 [客戶]，然後從清單中選擇客戶。

4. 在客戶的詳細資料頁面上選取 [軟體]，以查看為客戶購買的軟體清單。 

5. 找出您要取消的軟體購買，然後選取 [取消]。 此時會出現一個對話方塊。

6. 在 [訂單號碼] 下拉式清單中，選取您要取消的正確訂單識別碼。 (您可以在客戶的 [訂單歷程記錄] 頁面中深入了解訂單或訂單識別碼的詳細資訊。)

7. 選取核取方塊以確認您已閱讀關於取消的**重要**訊息。 (請參閱下列**重要**注意事項，以深入了解取消購買的後續事務)。

8. 選取 [提交] 以取消購買。 如果您想要取消客戶的多個訂單，您必須對每個唯一的訂單識別碼再次執行步驟 4 到 6。

當您嘗試取消訂單時，合作夥伴中心也可能會為您提供其他資訊 (顯示在 [訂單號碼] 下拉式清單下方)。 這類資訊可能包括：

- 您還剩下多少天可取消該訂單

- 您是否已超過取消時間範圍，而已無法再取消訂單

- 如果我們需要您提供更多關於取消要求的資訊，您可能會收到**客戶支援要求**表單的連結。

>[!IMPORTANT]
>在您取消訂單後，將會出現確認取消的訊息。 不過，最多可能需經過 15 分鐘的延遲，取消的訂單才會顯示在合作夥伴中心儀表板上。 

### <a name="post-cancellation-details"></a>取消後的詳細資料

在您取消購買後：

- 所有相關的軟體金鑰和下載連結都會撤銷。 這表示您或客戶都不能再使用與此購買有關的軟體金鑰和下載連結。 您和客戶須負責停止使用所有已取消的軟體。 您也必須負責解除安裝已取消的軟體，並移除任何相關的軟體下載和連結。

- 已取消的項目仍會顯示在客戶的軟體詳細資料頁面上，但啟用金鑰將無法使用。

- 對於已取消的訂單，其點數將會出現在您下個月的發票上。 永久軟體會收到 100% 的點數，而軟體訂閱則會收到按比例計算的點數。

### <a name="submit-a-customer-support-request-to-cancel-a-purchase"></a>提交客戶支援要求以取消購買

如果您嘗試透過合作夥伴中心來取消軟體購買，但被告知需提供更多資訊，並填寫客戶支援要求表單，下列步驟可能有所幫助：

1. 當您從 [取消購買] 視窗選取 [客戶支援要求] 連結時，將會開啟 [回報合作夥伴中心相關問題] 頁面。

2. 在 [詳細資料] 下的 [問題類型] 清單中，選取 [代表客戶的雲端解決方案提供者購買/退款]。

3. 填寫 [影響] 和 [標題] 欄位。

4. 在 [描述] 欄位中，提供下列各項：

    - 客戶租用戶 GUID 或網域名稱
    
    - 訂單識別碼或訂閱識別碼
    
    - 退款原因

    - 要求金額

5. 在 [連絡人] 欄位中，輸入您的名稱、電子郵件地址及電話號碼。

6. 如果您因故需附加檔案，請選取 [新增檔案]。 這是選擇性步驟。

7. 完成時，選取 [提交]。
