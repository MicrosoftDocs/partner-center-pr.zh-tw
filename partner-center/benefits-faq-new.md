# <a name="microsoft-partner-network-benefits-faq"></a>Microsoft 合作夥伴網路權益常見問題

**適當的角色**

- MPN 合作夥伴系統管理員

下列文章將回答 Microsoft 合作夥伴網路 (MPN) 合作夥伴關於其 MPN 成員資格權益、專長認證和產品授權的一些主要問題。

## <a name="responding-to-covid-19"></a>回應 COVID-19-19

**問**：我們的專長  **認證週年日即將推出，由於與 covid-19-19 相關的影響，我們無法滿足專長認證需求。我們該怎麼做？**

A. 如果您的專長認證週年日介於2021年1月1日到6月30日之間，我們將擴充您現有的專長認證或專長認證 (除了雲端 Business Applications 專長認證) 在2022年的下一週年日為止。 您將不需要符合測驗或效能需求，但仍需支付專長認證費用。如需詳細資訊，請參閱我們的 [blog 文章](https://blogs.partner.microsoft.com/mpn/responding-to-covid-19-microsoft-partner-network/)。

## <a name="membership"></a>成員資格

**問：更新專長認證的程式和時機為何？**

答：您的公司可以更新合作夥伴中心中的專長認證。 您需要符合專長認證需求、驗證每年所需的測驗和技能，以及在續約期間支付年度費用，以保留專長認證狀態和相關權益的存取權。

 在續約您的專長認證之前，貴公司的全域管理員或帳戶管理員需要審查和更新商務資訊。 他們可以存取合作夥伴中心中的公司設定檔來查看狀態，並檢查哪些資訊需要更新。 若要驗證您的資訊，可能需要其他檔。  

 您的訂閱續約時間範圍會從您購買日期的週年日加 1 天開始算起。 比方說，如果您在2020年9月1日購買了 Microsoft 行動套件，30天的續約期限將于2021年9月2日開始。 貴公司的全域管理員和 MPN 系統管理員角色，會收到最高達週年日和30天后的更新通訊。 您將在 [更新] 視窗中繼續存取相關聯的權益。

 如需其他資訊，請造訪 [更新 MPN 供應](renew-mpn-offers.md)專案。

**問：何時可以續訂我的 Microsoft 行動套件？**

答：您的公司可以在更新期間合作夥伴中心更新您的 Microsoft 行動套件。

在續約您的行動套件之前，貴公司的全域管理員或帳戶管理員將需要檢查並更新公司商務資訊。 他們可以在合作夥伴中心中存取您公司的設定檔，以查看您的狀態，並瞭解需要更新哪些資訊。 若要驗證您的資訊，可能需要其他檔。  

 您的訂閱續約時間範圍會從您購買日期的週年日加 1 天開始算起。 比方說，如果您在2020年9月1日購買了 Microsoft 行動套件，30天的續約期限將于2021年9月2日開始。您可在續約期間繼續存取相關權益。

 如需其他資訊，請造訪[更新 MPN 供應](renew-mpn-offers.md)專案

**問： Microsoft 行動套件與學習行動套件之間有何差異？**

答： Microsoft 行動套件是一系列的資源，可協助您熟悉 Microsoft 技術並打造您的解決方案/服務。 無論您是否有專長認證，都可以購買一個 Microsoft 行動套件。

學習行動套件是學習夥伴計畫的進入層級。 學習計劃為提供技術訓練及/或學習實務的企業提供資源。 [學習行動套件](https://partner.microsoft.com/asset/collection/learning-option-enrollment#/)提供有關課程、Marketplace 和權益的折扣，以協助合作夥伴成為各種 Microsoft 產品線的專家。
請參閱 Microsoft 行動套件和學習行動套件的 [授權表格](https://assetsprod.microsoft.com/microsoft-action-pack-license-table.pdf) 。

**問：** **30 天續約期間的專長認證或行動套件權益會發生什麼事？**

答：根據 MPN 合約，只要您的成員資格為使用中，您就可以繼續存取方案權益。 成員資格到期後，權益的存取將會遺失。

 如需其他資訊，請造訪[更新 MPN 供應](renew-mpn-offers.md)專案

## <a name="product-licenses"></a>產品授權

**問：產品授權 Microsoft 合作夥伴網路中的合作夥伴目前提供哪些供應專案？**

答：產品授權 (也稱為「內部使用權授權」或「內部使用權利」，) 包含在獲得專長認證或購買 Microsoft 行動套件的合作夥伴權益中。 從 [Microsoft 合作夥伴網路計劃指南](https://aka.ms/partner-benefits-use-guide)取得產品授權的目前供應專案清單。

**問：是否可以使用單一權杖來解除鎖定多個訂用帳戶？**

答：否，您無法分割權杖來解除鎖定多個訂閱。 標記必須使用提供的格式。 例如，25基座權杖無法分成5個不同的5基座權杖。

如需其他資訊，請參閱 Microsoft 合作夥伴計劃指南 [aka.ms/partner-benefits-use-guide](https://aka.ms/partner-benefits-use-guide)

**問：** **如果我們在錯誤的租使用者上啟用了我們的雲端服務權杖，或新增基座而不是擴充，我們可以取得更換權杖嗎？**

答：在原始權杖啟用之後的30天內，只能在30天內，于錯誤的租使用者上啟用雲端服務啟用的取代權杖，並在現有的訂用帳戶上不小心發出基座。

如需其他資訊，請參閱 Microsoft 合作夥伴計劃指南 [aka.ms/partner-benefits-use-guide](https://aka.ms/partner-benefits-use-guide)

**問：在內部部署軟體產品授權中，我們是否需要安裝產品金鑰？**

答：並非所有產品都需要產品金鑰。 Windows server 和用戶端作業系統以及所有 Microsoft Office 用戶端應用程式都需要啟用產品。

 如需其他資訊，請參閱 Microsoft 合作夥伴計劃指南 [aka.ms/partner-benefits-use-guide](https://aka.ms/partner-benefits-use-guide)

**問：如果我們使用了所有安裝和啟用功能，是否可以取得更多產品金鑰？**

答：發出的部分產品金鑰是 (MAK) 的多個啟用金鑰。 這些產品有有限數量的安裝、啟用或基座。 使用所有安裝和啟用時，產品金鑰將會停止運作。 系統將不會授與這些產品金鑰的其他啟用。

 如需其他資訊，請造訪 Microsoft 合作夥伴計劃指南 [aka.ms/partner-benefits-use-guide](https://aka.ms/partner-benefits-use-guide)

**問：如果我們決定不更新專長認證或 Microsoft 行動套件，是否仍可使用內部部署軟體產品授權？**

答：否。 專長認證與 Microsoft 行動套件所附的內部部署軟體產品授權並不是永久性的。
您有權取得許可權，而且只有在您的成員資格期間才會使用這些授權。 如果您選擇不續約您的專長認證或行動套件，您將需要透過商業授權取得授權或卸載任何透過 Microsoft 合作夥伴網路取得的內部部署軟體。

如需其他資訊，請造訪 Microsoft 合作夥伴計劃指南 [aka.ms/partner-benefits-use-guide](https://aka.ms/partner-benefits-use-guide)

**問： Microsoft 行動套件是否包含 Visual Studio Enterprise 訂用帳戶？**

答：否，Microsoft 行動套件只包含 Visual Studio Professional 訂閱。

如需其他資訊，請造訪 Microsoft 合作夥伴計劃指南 [aka.ms/partner-benefits-use-guide](https://aka.ms/partner-benefits-use-guide)

**問：我們可以分享專長認證或 Microsoft 行動套件所含的 Visual Studio 訂閱嗎？**

答：否，Visual Studio 的訂用帳戶是依使用者授權，而且必須指派和啟用才能有效。 禁止共用登入認證。

如需其他資訊，請造訪 Microsoft 合作夥伴計劃指南 [aka.ms/partner-benefits-use-guide](https://aka.ms/partner-benefits-use-guide)

**問：** **我們如何知道****公司有** 多少產品授權？

A. 您可以查看及管理您的專長認證或行動套件在合作夥伴中心中所包含的產品授權和其他權益。 深入瞭解如何 [管理您的合作夥伴網路權益](manage-your-partner-network-benefits.md)。

**問。我們的公司如何使用透過專長認證或行動套件提供給我們的產品授權？**

A. 我們包含 Microsoft 合作夥伴網路中取得一或多項專長認證或購買 Microsoft 行動套件的合作夥伴產品授權，以協助他們打造創新的解決方案，並熟悉 Microsoft 產品和技術。 專長認證和行動套件所包含的產品授權條款，將在 [Microsoft 合作夥伴網路計劃指南](https://aka.ms/partner-benefits-use-guide)中說明。

**問：** 我們 **的公司如何透過專長認證或行動套件，使用可供我們使用的產品授權****有何限制**？

A. 是的，有一些限制。 產品授權使用規定與透過 Microsoft 授權方案取得的商業授權不同。 如需詳細資訊，請參閱 [產品授權使用](https://www.microsoft.com/licensing)規定。
產品授權不包含軟體保證，而產品授權有一些限制。 您可以在《 [Microsoft 合作夥伴網路程式指南》](https://aka.ms/partner-benefits-use-guide)中檢查產品授權的使用條款。

**問：** **如何將更多產品授權新增至合作夥伴中心中列出的內容，以涵蓋我們的****公司？**

A. 專長認證和行動套件的產品授權，與透過 Microsoft 授權方案取得的商業授權不同的使用規定。 建議您在 [Microsoft 授權](https://www.microsoft.com/licensing)上審查您的選項，以符合貴公司的授權需求。

**問。我們最近在具有專長認證的 Microsoft 合作夥伴網路中，合併/取得了另一家公司。當我們在合作夥伴中心中使用帳戶合併功能時，是否要保留所有目前的產品授權？**

A. 專長認證的完成度和行動套件購買都是透過合作夥伴全球帳戶合作夥伴中心完成。 當您在合作夥伴中心中使用帳戶合併功能時，每個帳戶的現有權益都會保持原狀，直到下次更新為止。 在續約時，專長認證和行動套件所包含的權益會針對合作夥伴公司的合作夥伴全球帳戶實行。

-   Microsoft 行動套件：合作夥伴公司可能會針對每個合作夥伴全球帳戶購買一個行動套件。

-   專長認證：合作夥伴公司將獲得一份與最高達到效益相關的核心權益，以及合作夥伴可在合作夥伴全球帳戶中享有的專長認證權益。

 請參閱 [帳戶合併](merge-accounts.md) 的相關資訊，以深入瞭解此功能的運作方式，以及如何判斷它是否適合貴公司的需求。

**問：** **我們對****合作夥伴中心中的公司產品授權有疑問。我們可以在哪裡取得協助？**

A. 您可以移至 [合作夥伴中心的支援區段](https://partner.microsoft.com/support) ，以取得可能有所説明之資源的連結，或視需要向合作夥伴支援小組開啟票證。
